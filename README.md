# SAR Project

## Ramas

Cada vez que vayamos a añadir una nueva parte al proyecto debemos crear una nueva rama con el siguiente formato: ```feature/Nombre_de_la_rama,``` donde Nombre_de_la_rama deberá ser algún nombre descriptivo de la actividad que se quiere realizar.

Una vez hayamos finalizado la tarea haremos un pull requests desde nuestra rama a la rama dev que será la rama estable el proyecto en todo momento, cuando creemos el pull request debemos asignárselo a alguien para que al menos dos personas diferentes puedan comprobar el código.
